## Features

- Open Twitch home page
- Search for StarCraft II
- Open a streamer page
- Wait until the video loads
- Take a screenshot of the streamer page

## Installation

1. Clone this repository:
   git clone https://github.com/ricky8324/twitch-tests.git
   cd twitch-tests

2. Install dependencies:
   pip install -r requirements.txt
   (Make sure you have Python, Chrome installed and the correct ChromeDriver)

3. Running Tests:
   pytest tests/test_twitch.py


Screenshots will be saved automatically in the screenshots folder.
